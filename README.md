# 📄 ATS Resume Analyzer

An AI-powered ATS Resume Analyzer built using Streamlit and Ollama.

## 🚀 Features
- Upload resume (PDF)
- Paste job description
- ATS score calculation
- Keyword matching (rule-based)
- AI suggestions using Ollama
- Resume improvement using LLM

## 🛠️ Tech Stack
- Python
- Streamlit
- Ollama (Mistral)
- PyMuPDF

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
